問題を解決する

トラブルを解決するためには、まず「トラブル発生時のチェックリスト」を読んで原因の切り分け、状況の整理を行ってください。また、代表的なトラブルに対する対処法を参照するには、下のリンクをクリックしてください。

  • トラブル発生時のチェックリスト
  • 掲示板リストに何も表示されない
  • スレッド一覧を更新できない
  • 新着レスを取得できない
  • 書き込みできない
  • BathyScaphe が強制終了（クラッシュ）する
  • 特定の ID であぼーんしたいのに、うまくできない
  • 特定の ID を一覧検索したいのに、うまくできない
