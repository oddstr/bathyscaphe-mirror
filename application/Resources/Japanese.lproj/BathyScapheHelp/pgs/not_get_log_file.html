ログを取得できない／新着レスを取得できない

  • 一旦そのスレッドのログを削除してから、もう一度取得し直してみてください。「再取得が必要です」というアラートが表示された場合は、「削除して再取得」ボタンをクリックします。何も表示されない場合は、「スレッド」＞「ログを削除...」と選びます。
  • ２ちゃんねるが混雑しているのかもしれません。しばらく待ってからやり直してみてください。
  • サーバ移転かもしれません。掲示板リストを更新してみてください。
  • 新着レスが迷惑レスと判定されている可能性があります。迷惑レスの表示方法を「表示しない（透明あぼーん）」に設定している場合、見た目には何も表示されなくなります。

関連項目

  • 掲示板リストを更新する
  • 「フィルタ」を設定する
