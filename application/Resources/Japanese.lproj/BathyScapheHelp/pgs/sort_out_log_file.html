ログを整理する、dat 落ちしたスレッドを読む

dat 落ちしてスレッド一覧から消えてしまったスレッドでも、取得した分のログは残っています。こうしたログを見つけるためのいくつかのヒントを以下に示します：

  • Mac OS X v10.4 以降をお使いの場合、Spotlight を使用してログファイルを検索できます。BathyScaphe ログファイルからの検索に特化したアプリケーション「BSSpotlighter」を利用すると、さらに便利に検索できます。
    BSSpotlighter は、http://www.geocities.jp/svc9826/forBathyScaphe.html#BSSpotlighter から入手できます。
  • スマート掲示板を作成します。スマート掲示板の条件で「既得レス数」や「最終更新日」などを利用して、古いログを見つけ出すことができます。

重要：このバージョンの BathyScaphe では、「CMLogBuccaneer」や「CMLogFinder」を利用して dat 落ちログを正しく見つけ出すことができません。これらのツールはログファイルを一覧する目的にのみ使用することをおすすめします（BathyScaphe の今後のアップデートで、dat 落ちログを整理する手段を何らかの形で実装する予定です）。

重要：BSSpotlighter は BathyScaphe プロジェクトの一部ではありません。BSSpotlighter についての詳細は、BSSpotlighter の作者に直接お問い合わせください。
