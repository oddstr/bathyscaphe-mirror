ログを整理する、dat 落ちしたスレッドを読む

dat 落ちしてスレッド一覧から消えてしまったスレッドでも、取得した分のログは残っています。こうしたログを見つけるためのいくつかのヒントを以下に示します。

重要：このバージョンの BathyScaphe は、「CMLogBuccaneer」や「CMLogFinder」と互換性がありません。

参考：取得していない dat 落ちスレッドのログについては、●アカウントを購入することで取得できる場合があります。すでに●アカウントをお持ちの場合で、dat 落ちスレッドのログを取得する方法を知りたいときは、●で dat 落ちしたスレッドのログを取得する を参照してください。

  • Mac OS X v10.4 以降をお使いの場合、Spotlight を使用してログファイルを検索できます。BathyScaphe ログファイルからの検索に特化したアプリケーション「BSSpotlighter」を利用すると、さらに便利に検索できます。
    BSSpotlighter は、http://www.geocities.jp/svc9826/forBathyScaphe.html#BSSpotlighter から入手できます。
  • スマート掲示板を作成します。スマート掲示板の条件で「既得レス数」や「最終更新日」などを利用して、古いログを見つけ出すことができます。
  • dat 落ちしたスレッドのログが必要ない場合は、「掲示板」＞「dat 落ちしたスレッドのログを削除...」と選択します。

重要：BSSpotlighter や CMLogBuccaneer、CMLogFinder は BathyScaphe プロジェクトの一部ではありません。BathyScaphe 以外のソフトウェアについての詳細は、各ソフトウェアの作者に直接お問い合わせください。
