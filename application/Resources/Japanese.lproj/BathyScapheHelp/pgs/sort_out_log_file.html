ログを整理する、dat 落ちしたスレッドを読む

dat 落ちしてスレッド一覧から消えてしまったスレッドでも、取得した分のログは残っています。こうしたログを見つけたり、整理するには、専用のツール「CMLogFinder」や「CMLogBuccaneer」を使用すると便利です（詳しくはそれぞれのツールに付属するドキュメントをご覧ください）。

CMLogFinder や CMLogBuccaneer を使うには、それぞれのツールがインストールされていなければなりません。以下の Web サイトから入手してください。

CMLogFinder
http://homepage.mac.com/tsawada2/logfinder/
CMLogBuccaneer
http://homepage.mac.com/k_natori/Cocoa/

それぞれのツールがインストールされたら、ツールバーの「CMLogFinder」ボタンをクリックするか、「ウインドウ」メニューから「CMLogFinder」を選択することでツールを使用することができます。

  • BathyScapheから起動するログ管理ツールを、環境設定で変更することができます。詳しくは「詳細」を設定するをお読みください。
