名前欄、メール欄の内容をあらかじめ設定する

いつも同じ名前（コテハン）で書き込みたい、あるいはいつもメール欄に「sage」を入力したいときなど、名前欄とメール欄の内容をあらかじめ設定しておくことが可能です。

これらの設定は、「すべての掲示板」または「特定の掲示板」ごとに設定できます。さらに、掲示板ごとに Be ログインするかどうかも設定できます。

  • すべての掲示板で名前・メール欄の内容をあらかじめ設定するには、「書き込み」環境設定で設定します。詳細は「関連項目」のリンクをクリックしてください。
  • 掲示板ごとに名前欄・メール欄の内容を設定するには、「掲示板」＞「掲示板オプションを表示」と選択します。次に、名前欄とメール欄を入力します。

参考：掲示板ごとの設定は、すべての掲示板に対する設定よりも優先されます。

  • 掲示板ごとに Be ログインするかどうか設定するには、「掲示板」＞「掲示板オプションを表示」と選択します。次に、「Be ログインする」チェックボックスにチェックを入れます（あるいは、外します）。

参考：常に Be ログインが必要な掲示板や、２ちゃんねる以外の掲示板では、この設定は無視されます。

これらの設定内容は、書き込みウインドウに「あらかじめ入力される」だけであることに注意してください。名前欄・メール欄の内容や、Be ログインの設定は、必要に応じて実際の書き込み時に変更できます。
