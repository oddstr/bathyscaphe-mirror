「同期」を設定する

「同期」環境設定で、掲示板リストの同期に関する設定を行うことができます（以前のバージョンの「掲示板リスト・アップデート」または「BWAgent」の機能は、ここに統合されました）。

  • 「データの取得先：」から、同期したい BBSMenu.html の URL を選択するか、または直接 URL を入力します。
    注意：一部の Web サイトが提供している BBSMenu.html では、正しく同期できないことがあります。その場合は標準で用意されている取得先の中から選択してください。
  • 掲示板リストを自動的に一定間隔で同期させるには、「掲示板リストを自動的に同期：」にチェックマークを付け、ポップアップメニューから頻度を選択します。
  • 今すぐ手動で掲示板リストを同期させるには、「今すぐ同期」ボタンをクリックします。
