BathyScaphe ヘルプ

BathyScaphe を自分好みに変更する

フォント、スタイル、カラーをカスタマイズする

スレッド一覧やログ、書き込みウインドウ、レスポップアップなどのフォント、スタイル、カラーを細かく設定できます。

  1. アプリケーションメニュー（「BathyScaphe」メニュー）から「環境設定...」を選びます。
  2. 「表示」をクリックします。
  3. フォントやカラーを好みに設定します。

ツールバーをカスタマイズする

Finder や他の多くの Cocoa アプリケーションと同様に、ツールバーを自由にカスタマイズできます。

ツールバーをカスタマイズするには、「表示」＞「ツールバーをカスタマイズ...」と選びます。

Mail 欄の表示を設定する

Mail 欄は通常ポップアップで表示されますが、好みにより Mail 欄の内容を直接（インラインで）表示させることもできます。

  1. アプリケーションメニュー（「BathyScaphe」メニュー）から「環境設定...」を選びます。
  2. 「一般」をクリックします。
  3. 「メール欄を表示：」の「する」をクリックします。

外部リンクを開く Web ブラウザを変更する

リンクをクリックした際に起動する Web ブラウザは、Mac OS X が決定しています。これを変更するには次のようにします：

  1. Safari を起動します。
  2. 「Safari」メニューから、「環境設定...」を選び、さらに「一般」を選びます。
  3. デフォルトの Web ブラウザをポップアップメニューから選びます。

常時sageで書き込む

常にMail 欄にsageと書き込んでおきたい、あるいは常に同じ名前で書き込む人などは、標準の名前とMail 欄を設定しておくことができます。

  1. アプリケーションメニュー（「BathyScaphe」メニュー）から「環境設定...」を選びます。
  2. 「書き込み」をクリックします。
  3. 好きな名前やMail 欄を入力します。

ウインドウの位置とサイズを設定する

別ウインドウでスレッドを開いているとき、その位置とサイズはスレッドごとに記憶されますが、デフォルトの位置とサイズをあらかじめ決めておくことができます。

この設定を行うと、新たに開くスレッドのウインドウにデフォルトの位置とサイズが適用されるようになります。ただし、今まで開いたことのあるスレッドのウインドウには適用されません（それまでと同じ位置とサイズで表示されます）。

  1. 適当なスレッドを開き、ウインドウの位置とサイズを調節します。
  2. 「ウインドウ」＞「現在の位置とサイズを記憶」と選びます。
