リンクをプレビューする

BathyScaphe では、一部の外部リンクを内蔵のプレビューインスペクタでプレビューすることができます。内蔵のプレビューインスペクタは、主に画像ファイルのリンクをプレビューすることができます。

  • リンクをプレビューするには、単にそのリンクをクリックします。
  • プレビュー機能を使用したくない場合は、「詳細」環境設定で設定します。もっと詳しく知りたいときは、関連項目のリンクをクリックしてください。

リンクがプレビュー可能な場合は、プレビューインスペクタが開きます。そうでなければ、かわりに Web ブラウザでそのリンクが開かれます。

ヒント：プレビューインスペクタは「プラグイン」形式になっているため、他の開発者が作成したプラグインを別途インストールすることで、他の種類のリンクもプレビューしたり、あるいはより高機能なプレビューを行うことができるようになります。
