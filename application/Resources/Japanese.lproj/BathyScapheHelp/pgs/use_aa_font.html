AA フォントで表示する

Windows 環境での表示を前提に製作されている AA（アスキーアート）を、Mac OS X の環境でもズレなく表示することができます。

まず、モナーフォントなどの適切なフォントを入手する必要があります。詳しくは、BathyScaphe Web サイトを参照してください。

フォントをコンピュータにインストールした後、次のように設定します：

  1. 「BathyScaphe」メニューから「環境設定...」を選びます。
  2. 「表示」をクリックし、さらに「スレッドの表示」をクリックします。
  3. 「AA：」のフォントボタンをクリックし、「Mona」または「モナー」を選びます。12, 14, 16ポイントのいずれかをおすすめします。

AA フォントの適用は、手動で行う必要があります。

  • 特定のレスのみを AA フォントで表示させるには、レス番号をクリックして、コンテキストメニューから「AA」を選びます。
  • スレッドのすべてのレスを AA フォントで表示させるには、「スレッド」＞「印を付ける」サブメニューから「AA スレッド」を選んでチェックマークをつけます。

AA 系の掲示板などでは、あらかじめすべてのスレッドに「AA スレッド」の印を付けておきたいことがあります。その場合は「掲示板オプション」で設定します：

  1. 「掲示板」＞「掲示板オプションを表示」と選びます。
  2. 「デフォルトで AA スレッドにする」にチェックマークを付けます。

注意：「デフォルトで AA スレッドにする」オプションを選択する前に開いたスレッドについては、このオプションは反映されません。
