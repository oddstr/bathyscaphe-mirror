AA フォントで表示する

Windows 環境での表示を前提に製作されている AA（アスキーアート）を、Mac OS X の環境でもズレなく表示することができます。

まず、モナーフォント・MS Pゴシックフォントなどの適切なフォントを入手する必要があります。詳しくは、BathyScaphe Web サイトを参照してください。

フォントをコンピュータにインストール後、BathyScaphe で AA の表示にそのフォントを使うように設定します：

  • テーマを編集して、AA 用のフォントを指定します。または、あらかじめ AA 用のフォントが選択されたテーマに切り替えます。
    「テーマ」について詳しくは、関連項目のリンクをクリックしてください。

レスを AA フォントで表示させるには、次のような方法があります。

  • 「表示」環境設定で「自動的に AA フォントを適用する」にチェックを付けると、ほとんどの AA が自動的に AA フォントで表示されるようになります。
  • （手動で）特定のレスを AA フォントで表示させるには、レス番号をクリックして、コンテキストメニューから「AA」を選びます。
  • スレッドのすべてのレスを AA フォントで表示させるには、「スレッド」＞「印を付ける」サブメニューから「AA スレッド」を選んでチェックマークを付けます。

AA 系の掲示板などでは、あらかじめすべてのスレッドに「AA スレッド」の印を付けておきたいことがあります。その場合は「掲示板オプション」で設定します：

  1. 「掲示板」＞「掲示板オプションを表示」と選びます。
  2. 「デフォルトで AA スレッドにする」にチェックマークを付けます。

注意：「デフォルトで AA スレッドにする」オプションを選択する前に開いたスレッドについては、このオプションは反映されません。

AA フォントでの表示を解除する方法については、関連項目のリンクをクリックしてください。
