内容を検索する

BathyScaphe の検索機能を使って、表示中のスレッドからレス、名前、ID などを検索できます。検索結果を一覧表示することもできます。

  1. まず、検索したいスレッドを表示します。
  2. 「編集」＞「検索」＞「検索...」と選びます。
  3. 適切なオプションを選択して検索を実行します。
  • 「一覧検索」をクリックすると、検索語句を含むすべてのレスがポップアップで一覧表示されます。たとえば、特定の ID を「一覧検索」すると、その ID の全発言がポップアップ表示されます。
  • 「最初から検索」「前を検索」「後を検索」「すべて検索」は、いわゆる「省略されたレス」は検索対象になりませんので注意してください。「省略されたレス」も含めて検索したい場合は、「一覧検索」を利用するか、オンザフライボタンの「省略されたレス」ボタンをクリックして、すべてのレスを表示させた後に検索ボタンをクリックしてください。
