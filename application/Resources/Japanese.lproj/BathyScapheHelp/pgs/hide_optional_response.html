任意のレスをあぼーんする

迷惑レスフィルタをすり抜けた発言でも、コンテキストメニューから直ちにあぼーんすることができます。

  • レスの属性を「迷惑レス」「ローカルあぼーん」「透明あぼーん」のどれかに設定します。

設定方法、解除方法についてはレスの属性を設定するをお読みください。

「ローカルあぼーん」「透明あぼーん」については迷惑レスを表示しないようにするをお読みください。

「迷惑レス」を選択すると、そのレスは迷惑レスの「サンプル」として記憶されます。迷惑レスフィルタは、これらの「サンプル」を基に、迷惑レスと判断されたレスをすべて自動的にあぼーんします（迷惑レスフィルタについて詳しくは、下の「関連項目」を参照してください）。

一方、「ローカルあぼーん」「透明あぼーん」は、「迷惑レス」として記憶されることはありません。単に表示されなくなるだけです。

ヒント：特定の ID をあぼーんするには、その ID のレスを一つ選んで、迷惑レスに指定しましょう。詳細は迷惑レスフィルタを使うを参照してください。
