
これはCocoMonar version 1の開発に関するメモです。


[BUILD]

* Project Builder
--------------------------------
1. まず、cocomonar/frameworks以下の各フレームワークをビルドする。
2. いくつかのフレームワークはSGFoundation.frameworkをリンクしているため、最初にこれをビルドし、残りのフレームワークをビルドする。
3. cocomonar/application/CocoMonar.pbprojを開く。
4. ビルド・ターゲットを「v1」に変更
5. ビルド・アイコンをクリック

* Terminal
--------------------------------
1. cd cocomonar/
2. make v1



[ディレクトリ構造]

application/src/
	v1 : version1だけで使われているソース
	NG : version1, 2で共有されているソース
	More : ADCのSampleCodeから拾ってきたMoreUtilities
	test : テスト用ソース
