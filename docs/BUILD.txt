========================================
 HOW TO BUILD
========================================



■使用しているライブラリ

[sqlite]
--------------------------------
http://www.hwaci.com/sw/sqlite/

からダウンロードして、--enable-utf8でmakeし、スタティックライブラリにしたものを以下のディレクトリに置いてください。

cocomonar/frameworks/sqlite


[cURL]
--------------------------------
http://curl.haxx.se/libcurl/

./configure --prefix=(BuildPath) --without-ssl --without-zlib 
--disable-ipv6 --disable-shared --disable-thread --disable-nonblocking

また、configureしたあとで、lib/setup.hの27行目
#define HTTP_ONLY	
のコメントアウトをとり、
#include "config.h"したあとで、

#undef HAVE_STRTOK_R
#undef HAVE_STRLCAT

しています。


■ビルド

* Terminal
--------------------------------
1. cd cocomonar/
2. make


* Project Builder
--------------------------------
1. まず、cocomonar/frameworks以下の各フレームワークをビルドする。
2. いくつかのフレームワークはSGFoundation.frameworkをリンクしているため、最初にこれをビルドし、残りのフレームワークをビルドする。
3. cocomonar/application/CocoMonar.pbprojを開き、ビルドする。
