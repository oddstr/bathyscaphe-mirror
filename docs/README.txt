CocoMonar
--------------------------------

　CocoMonarは「巨大掲示板２ちゃんねる」を閲覧するための専用ブラウザです。
　ただし、Mac OS Xでしか動きません（動作確認：Mac OS X 10.2）。
　また、Mac OS Xで動作する2ch専用ブラウザとして他にiTteyoshiがあり、CocoMonarはこのブラウザを基に開発されています。

  ２ちゃんねる
    http://www.2ch.net/
  iTteyoshi
    http://www.geocities.co.jp/SiliconValley-Sunnyvale/9264/
